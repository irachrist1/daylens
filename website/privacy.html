Daylens Privacy Policy

Daylens is designed so your activity history remains under your control.

No data collection

Daylens does not collect personal data from users and does not transmit telemetry, analytics, or usage metrics to the developer.

Local-only tracking data

All app usage, browser history imports, website visit data, summaries, and local settings stay on your Mac in the Daylens application support directory.

No telemetry, analytics, or cloud sync

Daylens does not include background analytics SDKs, cloud account systems, or automatic sync services.

API key storage

If you configure an API key for AI features, it is stored in the macOS Keychain only.

User control

You can delete all Daylens data from Settings at any time. Removing the app also removes your ability to access any remaining local data unless you keep the application support directory.

Third-party AI requests

AI features are user-initiated. When you request a summary or ask a question that uses a configured provider, the relevant prompt and usage context are sent only to that provider for the requested response.
